smtp_auth

syntax: smtp_auth [login] [plain] [cram-md5] ;

default: login plain

context: main, server

With this directive you can set the permitted methods of authentication for SMTP clients:

  • login - AUTH LOGIN
  • plain - AUTH PLAIN
  • cram-md5 - AUTH CRAM-MD5
  • none - Disable SMTP Auth

Module: MailCoreModule
