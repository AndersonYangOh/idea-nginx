Directive can have multiple meanings. Each variant is separated with horizontal line

syntax: ssl_session_timeout time

default: ssl_session_timeout 5m

context: main, server

Assigns the time during which the client can repeatedly use the parameters of the session, which is stored in the cache.


Module: NginxHttpSslModule

syntax: ssl_session_timeout time

default: 5m

context: mail, server

Assigns the time during which the client can repeatedly use the parameters of the session, which is stored in the cache.


Module: NginxMailSslModule
