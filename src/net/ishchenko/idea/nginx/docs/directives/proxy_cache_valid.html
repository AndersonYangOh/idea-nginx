syntax: proxy_cache_valid reply

default: None

context: http, server, location

This directive set the time for caching different replies. Example: 

  proxy_cache_valid  200 302  10m;
  proxy_cache_valid  404      1m;

set 10 minutes cache time for replies with

If defined only time: 

  proxy_cache_valid 5m;

than only replies with codes 200, 301 and 302 will be cached.

Also it is possible to cache any replies with parameter "any": 

  proxy_cache_valid  200 302 10m;
  proxy_cache_valid  301 1h;
  proxy_cache_valid  any 1m;

Module: NginxHttpProxyModule
