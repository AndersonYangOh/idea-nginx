syntax: fastcgi_buffers the_number is_size;

default: fastcgi_buffers 8 4k/8k;

context: http, server, location

This directive sets the number and the size of buffers, into which will be read the answer, obtained from the fastcgi server. By default, the size of one buffer is equal to the size of page. Depending on platform this is either 4K, 8K or 16K. You can find out your page size by issuing the following command: 

getconf PAGESIZE

or 

getconf PAGE_SIZE

Or by compiling the following code: 

#include <unistd.h>
#include <stdio.h>
int main()
{
  printf("Page size on your system = %i bytes\n", getpagesize());
  return 0;
}
